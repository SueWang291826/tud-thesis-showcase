🗺️ Navigation Graph Construction

From raw IFC floor polygons to a 2.5D multi-level NetworkX graph with realistic traversal costs — fully automated without manual walkability labelling.

02 BIM Geometry Extraction

Three IFC files are parsed with ifcopenshell; spatial elements are converted to Shapely geometry objects.

IFC Files → Geometric Artefacts

IFC FileFloorKey Elements Extracted
站台层.ifc F1 Floor slab polygon, PSD doors, platform edges, obstacle walls
站厅层.ifc F3 Concourse polygon, fare gates, retail barriers, stair wells
设备层.ifc F2 Connector shaft geometry (reference only — no walkable nodes)

Extraction Pipeline

1 IfcSlab → floor polygon

Load all IfcSlab components; compute Shapely convex_hull / unary_union of their projected footprints to obtain the walkable boundary polygon.

2 IfcBuildingElement → obstacles (AABB)

Walls, columns, and room dividers are projected to 2D axis-aligned bounding boxes (AABB). Merged with the hand-calibrated obstacles_recalibrated.csv overlay.

3 IfcDoor → Platform Screen Doors

Doors on the platform face are classified as PSDs. Default state: closed. They are opened/closed in batch during ABM train-arrival events.

4 connectors_validated.csv → connector endpoints

Each row specifies {type, level_from, level_to, x_from, y_from, x_to, y_to}. These exact coordinates are used to create cross-level graph edges — no spatial inference needed.

Why 0.5 m? Sub-metre resolution captures corridor widths accurately (minimum ~1.5 m clear width), provides sufficient density for smooth paths, yet keeps the graph size tractable for real-time agent queries (~50 ms Dijkstra on full graph).

04 Vertical Connectors

Three connector types create cross-level edges with distinct travel-time profiles.

🪜
Staircase
Fixed vertical speed (ascent: 0.5 m/s, descent: 0.6 m/s). No capacity limit. Most numerous connector type. Used in baseline and dynamic scenarios.
bidirectional speed-adjusted cost
🛗
Escalator
Higher throughput; directional (single direction per unit). Travel time = floor height / (walk speed + belt speed). Direction toggleable in dynamic scenario.
directional belt speed bonus
🛗
Lift / Elevator
Low capacity (8 persons). Dwell time + travel time model. Required for Scenario C (wheelchair accessibility). Available on all four levels (F1–F4).
capacity-limited dwell model all levels
